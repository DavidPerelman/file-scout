# File Scout 🔍

אפליקציית Desktop לניהול וסידור קבצים במחשב — בעברית, מהירה, ובטוחה.

---

## למה File Scout?

קבצים מפוזרים בתיקיות שונות, כפילויות שתופסות מקום, אין תמונה ברורה של מה יש במחשב — File Scout פותרת את הכל במקום אחד.

---

## פיצ'רים

- 📁 **סריקת תיקיות** — סריקה רקורסיבית מלאה עם progress bar באחוזים
- 🔍 **פילטרים חכמים** — סינון לפי סוג קובץ, גודל מינימלי, ותאריך שינוי
- 👯 **זיהוי כפילויות** — מזהה קבצים זהים לפי תוכן (SHA-256) ולא רק לפי שם
- ✅ **בחירה חכמה** — כפתור "בחר כפולים" משאיר עותק אחד ומסמן את השאר למחיקה
- 🗑️ **מחיקה בטוחה** — שולח ל-Recycle Bin בלבד, לא מחיקה קבועה
- 🖱️ **פתיחת קבצים** — דאבל-קליק על שורה פותח את הקובץ בתוכנה המתאימה
- 🇮🇱 **ממשק בעברית** — תמיכה מלאה ב-RTL

## התקנה

```bash
# שכפל את הפרויקט
git clone <repo-url>
cd file-scout

# צור סביבה וירטואלית
python -m venv venv
venv\Scripts\activate  # Windows
# source venv/bin/activate  # Mac/Linux

# התקן תלויות
pip install -r requirements.txt
```

## מבנה הפרויקט

```
file-scout/
├── main.py          ← חלון ראשי, UI, וחיבור בין הרכיבים
├── scanner.py       ← סריקת קבצים רקורסיבית עם pathlib
├── duplicates.py    ← זיהוי כפילויות עם SHA-256 hashing
└── requirements.txt ← תלויות הפרויקט
```
